# GUNSTORE Demo

Project bài tập lớn môn Cơ sở Lập trình Web, xây dựng bằng HTML, CSS, JavaScript thuần và `localStorage`.

## Chức năng chính

- Đăng nhập, đăng ký, thông tin cá nhân
- Storefront sản phẩm, giỏ hàng, voucher, lịch sử mua hàng
- Quản trị sản phẩm, khách hàng, nhân sự, nhà cung cấp, khuyến mãi
- Báo cáo giao dịch và doanh thu

## Tài khoản demo

- Admin: `admin123 / 12`
- Khách hàng: `khach123@gmail.com / 12`

## Cách chạy

1. Mở file [login.html](./login.html)
2. Đăng nhập bằng tài khoản demo hoặc tạo tài khoản mới
3. Dữ liệu được lưu trong `localStorage` của trình duyệt

## Ghi chú

- Project đã được bổ sung ảnh placeholder nội bộ cho sản phẩm nên không cần ảnh thật
- Hero media trang chủ đã được đổi sang SVG nội bộ để tránh lỗi file `.mp4` bị thiếu
- Nếu muốn reset dữ liệu demo, hãy xóa `localStorage` của trình duyệt
